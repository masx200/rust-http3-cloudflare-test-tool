§allow_axfr: Option<bool>

Allow AXFR (TODO: need auth)

§stores: Vec<ServerStoreConfig>

Store configurations. Note: we specify a default handler to get a Vec containing a StoreConfig::Default, which is used for authoritative file-based zones and legacy sqlite configurations. #[serde(default)] cannot be used, because it will invoke Default for Vec, i.e., an empty Vec and we cannot implement Default for StoreConfig and return a Vec. The custom visitor is used to handle map (single store) or sequence (chained store) configurations.
