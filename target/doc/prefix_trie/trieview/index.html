Module trieview 

Source
Expand description

A TrieView (or a TrieViewMut) is a pointer to a specific element in a PrefixTrie, representing the sub-tree rooted at that node.

This module allows you to perform Set operations (union, intersection, difference) on PrefixMaps and PrefixSets, optionally of only a trie-view.
