Expand description

Trait that defines a joint prefix, for instance, one that describes either a IPv4 or IPv6 address.

The joint prefix is implemented for ipnet::IpNet, [ipnetwork::IpNetwork], and [cidr::IpCidr]. It is also implemented for either::Either, for which the Left and Right types both implement crate::Prefix.
